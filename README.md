# Wilms Tumor Pipelines & Analysis

Scripts to determine copy number and do variant calling for Wilms tumor sample (ref. genome hg19) on the DKFZ odcf cluster.

**Copy Number:**  Runs HMF *Amber &rarr;  Cobalt &rarr; Purple*

**Variant Calling:** Runs *mutect2 and strelka &rarr; takes intersection*
